# Node.js 网络

## OSI

- OSI 7层模型，open system interconnection 开放式系统互联

- 应用层
    - m面向用户提供服务
    - http 网页
    - SMTP 邮件
    - 文件传输协议 FTP
    - DNS 域名解析
    - 数据单元 报文
- 表示层
    - 数据格式处理
    - 数据编码
    - 数据的加密 SSL TLS 
    - 压缩，解压缩
- 会话层
    - 两个实体 建立使用链接的方法
    - 会话的管理，
    - 流量的控制
    - 数据寻址
    - 出错处理
- 传输层 连接的基础TCP、UDP
    - 传输数据管理
    - 处理传输过程差错
- 网络层 newwork layer
    - 寻址 不同网络之间的数据包 IP
    - 路由算法
- 数据链路层
    - 负责建立和管理节点之间的链路
    - 接收物理层数据帧
- 物理层
    - 实现计算机节点之间 比特流传递
    - 光纤、ethernet、无线
