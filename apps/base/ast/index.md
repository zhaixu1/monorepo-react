## 什么是前端工程化

 系统化、模块化、规范化的开发模式 =》 提升开发效率、提升开发质量、提升开发体验 =》 提升交付效率、质量、稳定性。

 - 业务规模不断壮大
 - 团队成员不断增加
 - 性能、体验要求
 - 团队成员对代码质量有要求


 ### 关键技术

 - 模块化开发 esm cjs 等规范。  bludless
 - 组件化开发 =》 组件库
 - 代码规范，质量保证 =》 eslint、 prettier、husky
 - 自动化构建 =》 webpack、rollup、vite、esbuild、swc 
 - 自动化测试 
    - jest vitest
 - 持续集成 CI/CD
     - github action
     - jenkins
