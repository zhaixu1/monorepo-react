# MyExpress 企业级框架 - 快速上手指南

## 📖 项目简介

MyExpress 是一个功能完整的企业级 Express.js 框架，专为现代 Web 应用程序开发而设计。它集成了身份验证、授权、数据库操作、缓存、日志记录、API 文档等企业级应用所需的全部功能。

## ⚡ 5分钟快速上手

### 1. 环境准备
确保您的系统已安装：
- Node.js 18+
- npm 或 yarn

### 2. 安装项目
```bash
cd apps/myexpress
npm install
```

### 3. 配置环境
```bash
# 复制环境变量模板
cp .env.example .env

# 编辑 .env 文件，配置以下必要参数：
# JWT_SECRET=your-super-secret-jwt-key
# DATABASE_URL=postgresql://username:password@localhost:5432/myexpress_db
```

### 4. 启动应用
```bash
# 开发模式启动
npm run dev
```

### 5. 访问应用
- 🌐 应用程序: http://localhost:3000
- 📚 API 文档: http://localhost:3000/api-docs
- ❤️ 健康检查: http://localhost:3000/api/health

## 🔥 核心功能预览

### 用户注册示例
```bash
curl -X POST http://localhost:3000/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "username": "testuser",
    "password": "Password123",
    "firstName": "测试",
    "lastName": "用户"
  }'
```

### 用户登录示例
```bash
curl -X POST http://localhost:3000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "password": "Password123"
  }'
```

### 获取用户信息
```bash
curl -X GET http://localhost:3000/api/auth/me \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

## 🐳 Docker 一键部署

如果您偏好使用 Docker：

```bash
# 启动所有服务（应用、数据库、Redis）
docker-compose up -d

# 运行数据库迁移
docker-compose exec app npx prisma migrate dev

# 查看应用状态
docker-compose ps
```

## 📋 主要 API 接口

| 接口 | 方法 | 描述 | 权限要求 |
|------|------|------|----------|
| `/api/auth/register` | POST | 用户注册 | 无 |
| `/api/auth/login` | POST | 用户登录 | 无 |
| `/api/auth/me` | GET | 获取个人信息 | 需要登录 |
| `/api/users` | GET | 获取用户列表 | 管理员 |
| `/api/posts` | GET | 获取文章列表 | 无 |
| `/api/posts` | POST | 创建文章 | 需要登录 |
| `/api/health` | GET | 健康检查 | 无 |

## 🔧 常见问题

### Q: 如何修改端口？
A: 编辑 `.env` 文件中的 `PORT=3000`

### Q: 如何连接数据库？
A: 配置 `.env` 文件中的 `DATABASE_URL`

### Q: 如何查看 API 文档？
A: 启动应用后访问 http://localhost:3000/api-docs

### Q: 如何添加新的 API 接口？
A:
1. 在 `src/routes/` 创建路由文件
2. 在 `src/services/` 添加业务逻辑
3. 添加相应的测试文件
