# useDebounce

## API

### 使用方法
const debouncedValue = useDebounce(value, 500);

| 参数         | 说明           | 类型         | 默认值   |
| ------------ | -------------- | ------------ | -------- |
| value | 不需要实时显示的值 | - | - |
| delay | 延迟时间| number | 1000 （单位ms）|
