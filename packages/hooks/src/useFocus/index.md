# useFocus

## API 

### `const [eleRef, setFocusFn] = useFocus()`

| 参数         | 说明           | 类型         | 默认值   |
| ------------ | -------------- | ------------ | -------- |
| 无 |  | - | - |


## 返回值
```js
 const [eleRef, setFocusFn] = useFocus();
```
- eleRef: 获取焦点的元素
- setFocusFn: 设置焦点的方法
